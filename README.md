## Project Overview

<br />



<p>
SkoolHub is an education management system that provides
</p>
<ul>
<li>A secure, seamless 2FA login experience</li>
 <img src="./photos/snip1.gif" alt="Login feature" width="300px" height="300px" />

 <img src="./photos/snipp2.gif" alt="Admin feature" width="300px" height="auto"/>

 <img src="./photos/snipp3.gif" alt="Assignments feature" width="300px" height="auto"/>

## Getting Started
<p>
    Instructions to setup Skoolhub on your local machine below.
</p>

### Prerequisites

[![NPM](https://img.shields.io/badge/NPM-%23000000.svg?style=for-the-badge&logo=npm&logoColor=white)](https://www.npmjs.com/)

```sh
npm install npm@latest -g
```

### Installation

1. Clone the repo
```sh
git clone https://github.com/Mecha-Godzilla-Source-Code/Skoolhub
```

2. Install dependencies using
```sh
npm install
```
3. Open two separate terminal tabs and run in your dev environment:
```
npm run server-dev
```
```
npm run client-dev
```
